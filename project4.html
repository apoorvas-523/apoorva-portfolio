3D-Printed Keychain

Beginner-Level CAD and Additive Manufacturing Project

← Back to Portfolio

Project Overview

This project demonstrates basic CAD modeling and 3D printing skills. The keychain was designed using Onshape and produced using additive manufacturing techniques.

Design Objectives

  • Create a keychain with an aesthetically pleasing shape
  • Ensure dimensions and tolerances are compatible with 3D printing
  • Optimize material usage for lightweight design
  • Demonstrate beginner-level CAD modeling and rendering skills

CAD Design

  • Software: Onshape
  • Features: 3D extrusions, fillets, engraving text
  • Export: STL file ready for slicing
  • Printing Time: ~50 mins

3D Printing Process

  • Printer: Monoprice Voxel printer with PLA material
  • Layer height: 0.2 mm
  • Print orientation optimized for strength and surface finish
  • Post-processing: Light sanding for smooth surface

The project demonstrates understanding of basic additive manufacturing workflow.

Skills Demonstrated

  • CAD design with Onshape
  • 3D modeling and part constraint application
  • Basic understanding of 3D printing technology
  • Attention to design-for-manufacturability principles
