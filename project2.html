Mind to Motion

EEG-Based Classification of Motor Imagery and Actual Hand Movements Using LSTM Models

Senior Project – Undergraduate | Published IEEE Paper

← Back to Portfolio

Clinical Motivation

Stroke and neurological injuries often impair voluntary hand movements. Non-invasive Brain-Computer Interfaces (BCI) using EEG signals can assist motor rehabilitation and control of prosthetic devices. This project explores differentiating motor imagery from actual hand movements using an LSTM-based EEG classification model.

System Architecture

The overall system pipeline for Mind to Motion:

  • EEG Acquisition (OpenBCI)
  • Signal Preprocessing & Artifact Removal
  • Feature Extraction
  • LSTM Model Classification
  • Output Interpretation for Motor Rehabilitation

Data Pipeline & Signal Processing

  • EEG signal acquisition from OpenBCI device
  • Preprocessing: Bandpass filtering and artifact removal using EEGLAB
  • Feature extraction: time-domain and frequency-domain features
  • Segmentation and windowing for LSTM input

LSTM Model Design

  • Built in MATLAB using sequential LSTM architecture
  • Trained to classify motor imagery vs actual hand movement
  • Loss function: cross-entropy; optimizer: Adam
  • Hyperparameters tuned for optimal EEG classification performance

Validation & Results

Performance of the LSTM model:

  • Motor Imagery vs Actual Hand Movement: 62.5% accuracy
  • Rest vs Actual Hand Movement: 72.5% accuracy

These results demonstrate the model's potential for BCI-assisted motor rehabilitation.
